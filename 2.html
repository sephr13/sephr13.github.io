راهنمای استفاده از ماژول helliemod

نصب ماژول

برای نصب ماژول، فقط کافیست دستور زیر را در ترمینال اجرا کنید:

pip install helliemod

وارد کردن در برنامه

پس از نصب، برای استفاده از توابع ماژول، آن را به شکل زیر وارد کنید:

import helliemod

توابع موجود در ماژول

  • fact(n) - محاسبه فاکتوریل عدد
  • compare(a, b) - بازگرداندن عدد بزرگ‌تر
  • digits(n) - شمارش تعداد ارقام عدد
  • sum_digits(n) - مجموع ارقام عدد
  • reverse(n) - مقلوب عدد
  • check_prime(n) - بررسی اول بودن عدد
  • lcm(a, b) - کوچک‌ترین مضرب مشترک
  • gcd(a, b) - بزرگ‌ترین مقسوم‌علیه مشترک
  • abso_value(n) - قدر مطلق عدد
  • fib_seq(n) - تولید دنباله فیبوناچی
  • sum_atob(a, b, step=1) - مجموع اعداد بین دو عدد با گام دلخواه

راهنما به صورت گرافیکی

برای مشاهده لیست توابع به صورت پنجره گرافیکی، می‌توانید از تابع helliemod.help() استفاده کنید.
