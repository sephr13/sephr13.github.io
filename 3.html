راهنمای استفاده از ماژول fruitmod

نصب ماژول

برای نصب ماژول، کافیست در ترمینال بنویسید:

pip install fruitmod

وارد کردن در برنامه

برای استفاده از توابع ماژول:

import fruitmod

توابع اصلی

  • sib(n) - محاسبه فاکتوریل
  • golabi(a, b) - مجموع اعداد بین دو عدد
  • tameshk(a, b) - عدد بزرگ‌تر
  • porteghal(n) - نزدیک‌ترین مربع کامل
  • talebi(n) - شمارش ارقام
  • havij(n) - مقلوب عدد
  • kivi(n) - بررسی عدد کامل
  • khiar(n) - بررسی اول بودن
  • holoo(a, b) - بزرگ‌ترین مقسوم‌علیه مشترک
  • ananas(n) - عدد nام دنباله فیبوناچی

راهنمای گرافیکی

برای دیدن لیست توابع به صورت گرافیکی از:

fruitmod.help()

استفاده کنید.
